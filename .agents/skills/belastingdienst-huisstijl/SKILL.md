# Belastingdienst Huisstijl

This is a local-only skill workspace for a broader Belastingdienst house-style reference set.

## Intended Scope

- house-style source references
- reusable design rules
- presentation, document, and visual-system guidance
- mappings from official style-guide concepts to reusable theme parts

## Local-Only Policy

- Do not place confidential source material in tracked files here.
- Use the subdirectories below for local research and bundled references while this repository remains on GitHub.
- The directory is git-ignored except for this `SKILL.md` and nested `.gitignore` files.

## Structure

- `references/pages/`
- `references/images/`
- `assets/`
- `scripts/`

`references/pages/` contains the copied local markdown pages from the huisstijl reference set.

`references/images/` contains the copied image assets referenced by that material.
