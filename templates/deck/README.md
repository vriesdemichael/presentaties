# Deck template

Kopieer deze map naar `presentations/<deck-naam>/` om een nieuwe presentatie te starten.

Deze template is bedoeld voor de huidige workspace/fork-opzet. De dependency naar het gedeelde theme loopt daarom nog via `workspace:*`.

Als de private registry later beschikbaar is, verandert alleen de dependency-bron. De deck-structuur blijft hetzelfde.

Pas daarna minimaal aan:

- `package.json` naam
- `slides.md` titel, metadata en inhoud
